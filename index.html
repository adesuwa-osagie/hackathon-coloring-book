About This Coloring Book

"Let's Paint: DanganRonpa" is a fan-made coloring booking app based on the Japenese visual novel franchise called Danganronpa, created by Kazutaka Kodaka and published by Spike Chunsoft.

The app was made for Mintbean's November 2021 Hiring Hackathon, using HTML, CSS, JavaScript, and jQuery.

Using this app, a user can drag a mouse across a canvas to make free-form lines.

Compatible with tablets and the desktop. Not compatible with mobile devices at this time. The app works in browsers compatible with Windows. Apple users, let me know if it works for you, too!


What Else Can This App Do?

  • Paint with a color with an opacity of their choice
  • Use a dropper to select a color on screen
  • Change the pen size
  • Use an eraser
  • Download painting
  • Destroy painting (with no hope of recovery)
